# Chromecast Helper
<b>Design Usage:</b><br>
Simple way to help keep your Goolge/Nest devices alive.<br><br>
<b>New Install:</b><br>
* Copy the code from GitHub into a ‘New App’ under the ‘Apps Code’ menu then click ‘Done’
* Go to ‘Apps’
* Click ‘Load New Apps’
* Select ‘Chromecast Helper’ under ‘User Apps’
