<b>New Install:</b><br>
Install App
* Copy the App code from GitHub into a ‘New App’ under the ‘Apps Code’ menu
* Click ‘Save’

Install Driver
* Copy the Driver code from GitHub into a ‘New Driver’ under the 'Driver Code' menu then click ‘Done’

Go to ‘Apps’
* Click ‘Load New Apps’
* Select ‘Gentle Wake Up’ under ‘User Apps’
